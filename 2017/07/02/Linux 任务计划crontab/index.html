crontab 的深入

每个用户使用 crontab -e添加计划任务，都会在/var/spool/cron/crontabs中添加一个该用户自己的任务文档，这样目的是为了隔离。

如果是系统级别的定时任务只需要以 sudo 权限编辑 /etc/crontab 文件就可以。

cron 服务监测时间最小单位是分钟，所以 cron 会每分钟去读取一次 /etc/crontab 与 /var/spool/cron/crontabs 里面的內容。
